# SSMS Olivine Theme

Color scheme for SQL Server Management Studio.

![Preview](https://github.com/mg0x7BE/ssms-olivine-color-theme/blob/master/readme_files/preview.gif)

## Installation

1. Download [SSMS_Olivine_Theme.vssettings](https://github.com/mg0x7BE/ssms-olivine-color-theme/blob/main/SSMS_Olivine_Theme.vssettings)
2. Open SSMS → Tools → Import and Export Settings

![Installation](https://github.com/mg0x7BE/ssms-olivine-color-theme/blob/master/readme_files/installation.gif)
